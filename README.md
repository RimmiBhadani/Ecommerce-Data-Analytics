# E-Commerce Data Analytics Project  

## Project Overview
This project analyzes e-commerce sales data using Databricks SQL and Power BI.
I built an end-to-end analytics solution including data cleaning,
fact & dimension modeling (star schema), and an interactive business dashboard
to help understand performance, trends and customer behavior.

## Tech Stack
- Databricks SQL
- Power BI
- SQL (ETL, Data Cleaning, Modeling)

## Data Processing Steps
1️. Loaded raw dataset into Databricks  
2️. Cleaned null values & handled duplicates  
3️. Created `fact_sales` table  
4️. Built dimension tables: date, product, customer, payment, priority  
5️. Connected Power BI to Databricks SQL Warehouse  
6️. Designed dashboard & KPIs  

## Dashboard Insights
- Total Sales, Profit, Orders, Quantity  
- Sales Trend by Month  
- Sales by Product Category  
- Sales by Payment Method  
- Sales by Device Type  
- Profit Margin
